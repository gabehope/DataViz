Law of total probability:

\[ p(x) = \sum_{k=1}^K p(x, z=k), \quad z \in S_z = \{1,...,K\}\] \[ p(x) = \int_{S_z} p(x, z)dz, \quad z \in S_z \subseteq \mathbb{R}\]

Bayes rule:

\[p(z\mid x) = \frac{p(x\mid z)p(z)}{p(x)}\]

Normal PDF \[p(x) = \mathcal{N}(x; \mu, \sigma) = \frac{1}{\sigma \sqrt{2 \pi}} e^{-\frac{1}{2}\left( \frac{x-\mu}{\sigma} \right)^2}\]

Normal PDF: \[p(x) = \mathcal{N}(x; \mu, \sigma) = \frac{1}{\sigma \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu}{\sigma} \right)^2\right)\]

Mixture of 2 Normals \[p(x) = p_1 \mathcal{N}(x; \mu_1, \sigma_1) + p_2 \mathcal{N}(x; \mu_2, \sigma_2)\]

\[p(x) = p_1 \cdot \frac{1}{\sigma_1 \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_1}{\sigma_1} \right)^2\right) + p_2 \cdot \frac{1}{\sigma_2 \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_2}{\sigma_2} \right)^2\right)\]

\[p_1 \geq 0,\ p_2 \geq 0, \quad p_1 + p_2 = 1\]

\[\therefore \quad \int p(x) dx = 1\]

Mixture of \(K\) Normals \[p(x) =\sum_{k=1}^Kp_k \mathcal{N}(x; \mu_k, \sigma_k) \]

\[p(x) =\sum_{k=1}^Kp_k \cdot \frac{1}{\sigma_k \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_k}{\sigma_k} \right)^2\right)\]

Gaussian KDE

\[\hat{p}(x) = \frac{1}{2Nh} \sum_{i=1}^N \phi\left(\frac{x - x_i}{h}\right), \quad \phi(u) = \frac{1}{\sqrt{2 \pi}} \exp\left(-\frac{u^2}{2} \right)\]

Sampling from a mixture

\[z\in \{1,...,K\}\]

\[p(z): \quad z \sim \text{Categorical}(p_1, p_2, ...,p_K)\] \[p(z=k) = p_k\]

\[p(x\mid z =k): \quad x \sim \mathcal{N}(x; \mu_k, \sigma_k)\]

\[p(x \mid z_i=k) = \frac{1}{\sigma_k \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_k}{\sigma_k} \right)^2\right)\]

def sample(p1, mu1, sigma1, mu2, sigma2):
    z = np.random.random() > p1
    if z:
        return norm(mu2, sigma2).rvs()

        return norm(mu1, sigma1).rvs()

\[p(x, z) = p_z \cdot \frac{1}{\sigma_z \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_z}{\sigma_z} \right)^2\right) = p(x\mid z)p(z)\]

def log_likelihood(x, p1, mu1, sigma1, mu2, sigma2):
    p2 = 1 - p1
    lik = p1 * norm(mu2, sigma2).pdf(x) + p2 * norm(mu1, sigma1).pdf(x)
    return np.log(lik).sum()

Choosing \(z_i\). \(z_i\) reflects which distribution \(x_i\) came from, so intuitively it should reflect, which distribution it was more likely to come from.

\[p(z_i = 1 \mid x_i) \overset{\text{?}}{>}\ p(z_i = 2 \mid x_i)\]

\[z_i = \underset{z}{\text{argmax}}\ p(z \mid x_i)\]

\[p(z_i = k \mid x_i) = \frac{p(x_i \mid z_i = k ) p(z_i=k)}{\sum_{c=1}^K p(x_i \mid z_i = k ) p(z_i=k)} = \frac{p_k \cdot \frac{1}{\sigma_z \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_k}{\sigma_k} \right)^2\right)}{ \sum_{c=1}^K p_c \cdot \frac{1}{\sigma_z \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x-\mu_c}{\sigma_c} \right)^2\right)}\]

This gives us a new partition of our data. We can estimate our parameters again and so on!

def assign(x, p1, mu1, sigma1, mu2, sigma2):
    p2 = 1. - p1
    p_x_given_1 = norm(mu1, sigma1).pdf(x)
    p_x_given_2 = norm(mu2, sigma2).pdf(x)

    p_x_and_1 = p_x_given_1 * p1
    p_x_and_2 = p_x_given_2 * p2

    p_1_given_x = p_x_and_1 / (p_x_and_1 + p_x_and_2)
    p_2_given_x = p_x_and_2 / (p_x_and_1 + p_x_and_2)

What \(I(z_i=1) \in \{0, 1\} \longrightarrow w_i \in [0, 1]\)

\[\mu_1, \sigma_1 = \underset{\mu_1, \sigma_1}{\text{argmax}} \sum_{i=1}^N I(z_i=1) \log p(x_i \mid z_i = 1)\] \[\mu_1 = \frac{1}{\sum_{i=1}^N I(z_i=1)} \sum_{i=1}^N I(z_i=1) x_i\] \[\sigma_1 = \sqrt{\frac{1}{\sum_{i=1}^N I(z_i=1)} \sum_{i=1}^N I(z_i=1) (x_i - \mu_1)^2}\] \[p_1 = \frac{\sum_{i=1}^N I(z_i=1)}{N}\]

\[\mu_1, \sigma_1 = \underset{\mu_1, \sigma_1}{\text{argmax}} \sum_{i=1}^N w_{i1} \log p(x_i \mid z_i = 1)\] \[\mu_1 = \frac{1}{\sum_{i=1}^N w_{i1}} \sum_{i=1}^N w_{i1} x_i\] \[\sigma_1 = \sqrt{\frac{1}{\sum_{i=1}^N w_{i1}} \sum_{i=1}^N w_{i1} (x_i - \mu_1)^2}\] \[p_1 = \frac{\sum_{i=1}^N w_{i1}}{N}\]

def maximize(x, w):
    w1, w2 = w
    mu1 = (w1 * x).sum() / w1.sum()
    sigma1 = np.sqrt((w1 * (x - mu1) ** 2 ).sum() / w1.sum())

    sigma2 = np.sqrt((w2 * (x - mu2) ** 2 ).sum() / w2.sum())

Expectation-Maximization algorithm (for Gaussian mixtures)

E-Step

For all \(i\in \{1,...,N\}\), \(k\in \{1,...,K \}\) set:

\[w_{ik} = p(z_i=k \mid x_i)\]

M-Step

For all \(k\in \{1,...,K \}\) set:

\[\mu_k = \frac{1}{\sum_{i=1}^N w_{ik}} \sum_{i=1}^N w_{ik} x_i\] \[\sigma_k = \sqrt{\frac{1}{\sum_{i=1}^N w_{ik}} \sum_{i=1}^N w_{ik} (x_i - \mu_k)^2}\]

Repeat!

def em(x, eps=1e-5):
    params = maximize(x, x > 5)

    old_llik = -np.inf
    llik = log_likelihood(x, *params)

    for _ in range(100):
        z = assign(x, *params)
        params = maximize(x, z)
        new_llik = llik

        if new_llik < old_llik + eps:
            break
    return params

\[\log \sum_{i=1}^N p(x) =\sum_{i=1}^N \log \left( \sum_{k=1}^K p(x_i\mid z_i=k)p(z_i=k) \right)\]

\[q(z_i) = \text{Categorical}(w_{i1},...,w_{iK})\] \[q(z_i=k) = w_{ik}\]

\[\log \sum_{i=1}^N p(x) = \sum_{i=1}^N \log \left( \sum_{k=1}^K \frac{q(z_i = k)}{q(z_i = k)} p(x_i\mid z_i=k)p(z_i=k) \right)\] \[= \sum_{i=1}^N \log \left( \sum_{k=1}^K q(z_i = k) \frac{p(x_i\mid z_i=k)p(z_i=k)}{q(z_i = k)} \right)\]

\[=\sum_{i=1}^N \log \mathbb{E}_{q(z_i)}\left[ \frac{p(x_i\mid z_i=k)p(z_i=k)}{q(z_i = k)} \right] \]

\[\geq \sum_{i=1}^N \mathbb{E}_{q(z_i)}\left[\log \frac{p(x_i\mid z_i=k)p(z_i=k)}{q(z_i = k)} \right] \]

If \(q(z_i = k) = p(z_i =k \mid x_i)\)

\[ \sum_{i=1}^N \mathbb{E}_{q(z_i)}\left[\log \frac{p(x_i\mid z_i=k)p(z_i=k)}{q(z_i = k)} \right] \]

\[= \sum_{i=1}^N \sum_{k=1}^K p(z_i =k \mid x_i) \log \left( \frac{p(x_i\mid z_i=k)p(z_i=k)}{p(z_i =k \mid x_i)} \right) \]

\[= \sum_{i=1}^N \sum_{k=1}^K p(z_i =k \mid x_i) \log \left( \frac{p(x_i, z_i=k)}{p(z_i =k \mid x_i)} \right) \]

\[= \sum_{i=1}^N \sum_{k=1}^K p(z_i =k \mid x_i) \log \left( \frac{p(z_i=k \mid x_i) p(x_i)}{p(z_i =k \mid x_i)} \right) \]

\[= \sum_{i=1}^N \sum_{k=1}^K p(z_i =k \mid x_i) \log p(x_i)\]

\[= \sum_{i=1}^N \log p(x_i) \sum_{k=1}^K p(z_i =k \mid x_i) \]

\[= \sum_{i=1}^N \log p(x_i) \]

Recall \(w_{ik} = q(z_i = k)\)

\[ \sum_{i=1}^N \mathbb{E}_{q(z_i)}\left[\log \frac{p(x_i\mid z_i=k)p(z_i=k)}{q(z_i = k)} \right] \]

\[= \sum_{i=1}^N \sum_{k=1}^K w_{ik} \log \left( \frac{1}{w_{ik}} p(x_i\mid z_i=k)p(z_i=k) \right)\]

\[= \sum_{i=1}^N \sum_{k=1}^K \left[ w_{ik} \log p(x_i\mid z_i=k) + w_{ik} \log p(z_i=k) - \log w_{ik} \right]\]

\[p(z_i=k) = p_k\]

\[p(x_i \mid z_i=k) = \frac{1}{\sigma_k \sqrt{2 \pi}} \exp\left(-\frac{1}{2}\left( \frac{x_i-\mu_k}{\sigma_k} \right)^2\right)\]

Maximizing over \(\mu_k\) only depends on

\[ \sum_{i=1}^N w_{ik} \log p(x_i\mid z_i=k)\]
