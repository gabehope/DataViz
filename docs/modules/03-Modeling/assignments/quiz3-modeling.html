Q3: Variations on K-means

The k-medians algorithm is a variation of the k-means algorithm. In k-means, the centroid of each cluster is computed by taking the mean value for each variable among all observations assigned to that cluster. In k-medians, the centroid is instead computed by taking the median of each variable.

The code below runs k-means on the penguins dataset. Modify the code to perform k-medians instead. Then answer the following questions by visualizing the resulting cluster assignments.

from sklearn.metrics import pairwise_distances
measurements_only = penguins[['bill_length_mm', 'bill_depth_mm']]

num_clusters = 3
np.random.seed(55)
measurements_with_clusters = measurements_only.copy()
measurements_with_clusters['cluster'] = np.random.randint(0, num_clusters, (measurements_only.shape[0],))

for iteration in range(10):
    centroids = measurements_with_clusters.groupby('cluster').mean()
    distances = pairwise_distances(measurements_only, centroids)
    measurements_with_clusters['cluster'] = distances.argmin(axis=1)


ggplot(measurements_with_clusters, aes(x='bill_length_mm', y='bill_depth_mm', shape=as_discrete('cluster', order=1), color=as_discrete('cluster', order=1))) + \
    geom_point() +\
    scale_color_brewer(palette="Set2")

Answer:

We can clearly see from the plot below that distribution A is the better fit.

x = np.linspace(0, 15, 1000)
y_A = 0.
for ind, row in A.iterrows():
    y_A = y_A + row['cluster_probability'] * norm(row['mean'], row['std']).pdf(np.linspace(0, 15, 1000))

y_B = 0.
for ind, row in B.iterrows():
    y_B = y_B + row['cluster_probability'] * norm(row['mean'], row['std']).pdf(np.linspace(0, 15, 1000))

ggplot(dataset, aes(x='value')) + geom_density(bw=0.15)  + geom_line(aes(x=x, y=y_A), color='red') + geom_line(aes(x=x, y=y_B), color='blue')
