Q1: Interpreting a dataset

Take a look at the given dataset. What does one row of this dataset represent?

df = data.seattle_weather()
df.head()

1 2012-01-02 10.9 10.6 2.8 4.5 rain

3 2012-01-04 20.3 12.2 5.6 4.7 rain
4 2012-01-05 1.3 8.9 2.8 6.1 rain

Answer:

Each row represent a single day of weather data (for Seattle), including the temperature, precipitation and wind.

How many nominal variables are present in this dataset?

Answer

We can see that there is 1 nominal variable (weather).

Answer
